#### 使用说明

1.  WeChatMessageUtils:微信消息加解密工具包
2.  WeChatComponent.py:微信第三方平台开发工具
3.  WeChatLogin.py:微信登录工具
4.  WeChatPayment.py:微信支付工具

 **注：微信支付和微信登录均以手机网站为例，电脑网站和app需要更换对应参数和请求地址**
